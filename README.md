## 📦 项目作品

### 🛒 电商销售系统
> 一个独立设计并开发的全栈电商平台，涵盖用户管理、商品展示、购物车、订单处理等核心功能。

- **技术亮点：**
  - **后端**：Spring Boot + MyBatis + Spring Security + JWT，构建安全稳定的 RESTful 接口。
  - **数据库**：合理设计 MySQL 表结构（用户、商品、订单、购物车），兼顾数据一致性与查询效率。
  - **前端**：Vue + Element-UI 打造响应式界面，提升用户体验。
  - **工程化**：分层架构（Controller-Service-Mapper），代码清晰易维护。
